Help

Further Information

This website was built using the Instagram API. It is intended to be a tool to allow users to organize Instagram accounts into groups. Groups can be created and named by the user. Search functionality is provided so that users can find specific accounts, or discover media of interest.

Search

There are four search functions offered on this site: username search, popular media, media tags, and location. These are all independent search functions meaning they are not intended to be used at the same time but rather for individual searches. The Username Search enables the ability to find specific Instagram accounts when a username is known. Search results will include the accounts with the closest match to the search terms provided. Clicking the "View" button under Popular Media will return the current top trending posts on Instagram. With Tags Search, media can be searched by the "tags" or subjects linked to media posts. Finally, Location Search can be used to view media that has been posted from a certain geographic area.

Manage Groups

This site provides functionality to create and manage groups. New groups can be added with the Create a Group button. Users can be added to groups with the Add to Group buttons that appear in the user and media search results. Once groups have been created, they are always visible on the left side of the page, and they can be viewed and managed with the View Group, Edit Name, and Delete Group icons.
